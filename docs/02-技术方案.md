## 1. 技术选型

| 层 | 技术 |
|---|------|
| 开发语言 | Kotlin（主）+ Java（少量兼容） |
| UI 框架 | Jetpack Compose + 传统 View（悬浮窗用 WindowManager） |
| 架构模式 | MVVM + Clean Architecture + 单 Activity 多 Compose 屏 |

| 网络 | Retrofit + OkHttp + WebSocket（流式 ASR） |
| 悬浮窗 | WindowManager + `TYPE_APPLICATION_OVERLAY`（API 26+） |
| DI | Hilt |
| 打包 | Gradle 8.x + AGP 8.x |
| 版本 | `minSdk` 26 / `targetSdk` 36 / `compileSdk` 36 |

## 3. 模块划分

### 3.1 悬浮窗模块

- 使用 `WindowManager.LayoutParams.TYPE_APPLICATION_OVERLAY`（API 26+）
- 标题栏 `onTouch` 实现拖动
- 4 角 + 4 边 handle 实现缩放
- 边界 clamp 到屏幕内、不遮挡状态栏

### 3.2 AI 智能滚动引擎

```
麦克风采集 PCM ──► AudioRecord ──► 分片(每 200ms)
                                       │
                                       ▼
                              WebSocket 推送至 MiniMax ASR
                                       │
                                       ▼
                              实时返回识别文本片段
                                       │
                                       ▼
                       MatchEngine 模糊匹配到文稿位置
                                       │
                                       ▼
                       FloatingView 滚动到该位置 + 高亮
```

### 3.3 滚动控制

- 手动：手指滑动
- 匀速：`Choreographer` 驱动，每帧根据 dt 计算 scrollBy
- AI 接管：`MatchEngine` 输出新位置 → `scrollTo` 平滑过渡

### 3.4 持久化

```kotlin
@Entity(tableName = "scripts")
data class ScriptEntity(
    @PrimaryKey val id: String,
    val title: String,
    val content: String,
    val updatedAt: Long,
    val tags: String
)

@Entity(tableName = "settings")
data class SettingEntity(
    val key: String,
    val value: String
)
// 包括：字体大小、透明度、颜色、滚动速度、最近文稿ID
// API Key 单独放 EncryptedSharedPreferences
```

## 6. 核心交互时序（AI 提词）

```
[User] 点击悬浮按钮
  ↓
[FloatingWindowService] 显示提词框 + 启动 PrompterEngineService
  ↓
[PrompterEngineService] 前台通知 + 启动 AudioRecorder
  ↓
[AudioRecorder] 持续推 PCM 至 Flow
  ↓
[AsrEngine] WebSocket 连接 MiniMax → 流式识别
  ↓
[MatchEngine] 接收识别片段 → 计算匹配位置
  ↓
[FloatingView] scrollTo(y) + 当前行高亮
  ↓
[User] 关闭悬浮 → Service stopForeground + 释放麦克风
```

## 7. 安全与隐私

| 风险 | 对策 |
|------|------|
| API Key 泄露 | EncryptedSharedPreferences (AES-256 GCM) |
| 麦克风滥用 | 悬浮窗运行时强制通知 + 权限再次确认 |
| 文稿泄露 | 默认仅本地；不上传；导出需用户主动操作 |
| 网络 | ASR 仅发送音频流，不上传文稿本身 |
| WebSocket 断连 | 自动重连 + 失败降级为本地匀速滚动 |
